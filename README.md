
# Student Marks Calculator

A Python application that calculates student academic 
performance across multiple subjects.

## Features
- Enter student name
- Enter marks for Mathematics, English and Science
- Automatically calculates average mark
- Assigns grade: Distinction (75%+), Pass (50%+) or Fail
- Displays personalised feedback message based on result

## Technologies Used
- Python 3

## How To Run
1. Download the file
2. Open in Python IDLE or any code editor
3. Run the program
4. Enter student name and subject marks
5. View results and grade
